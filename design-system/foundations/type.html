Rules

  • Headings get letter-spacing:-0.02em and text-wrap:balance.
  • Tabular numerals on every score, counter, and metric — numbers never jitter.
  • Mono (ui-monospace, 'IBM Plex Mono') is for numeric readouts only, never prose.
  • 11px uppercase labels are the only tracked type; body copy is never all-caps.
